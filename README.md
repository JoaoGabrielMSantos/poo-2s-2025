# AT3 - Projeto Final Java

Sistema de cadastro de pessoas (Alunos e Professores) usando Java e SQLite.

## Funcionalidades

- Cadastro de Alunos e Professores
- Consulta de todas as pessoas cadastradas

## Tecnologias

- Java 11
- SQLite
- Maven
- JDBC

## Como Executar

### Com Maven:
```bash
# Compilar
mvn clean compile

# Executar
mvn exec:java -Dexec.mainClass="Main"
